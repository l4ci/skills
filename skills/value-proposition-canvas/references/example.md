# Example: value-proposition-canvas

**You invoke it with:**
> "Build a Value Proposition Canvas for our app and tell me if it actually fits the customer."

**It needs from you:**
- VALUE PROPOSITION (required): PantryPilot, a meal-planning app that auto-generates grocery lists from saved recipes and syncs them to delivery services
- CUSTOMER SEGMENT (required): dual-income parents of young kids who cook on weeknights and hate planning
- CONTEXT (optional): early stage, 20 user interviews done, deciding whether to build the delivery-sync feature or pivot

**You get back:** a saved markdown report (`value-proposition-canvas-<subject-slug>-<date>.md`) with the ranked Customer Profile, the Value Map tied to it, a fit-check table (hit/partial/miss), flagged irrelevant relievers and creators, the fit level reached, and the riskiest assumptions to test first.
