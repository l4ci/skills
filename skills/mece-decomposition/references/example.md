# Example: mece-decomposition

**You invoke it with:**
> "Build me an issue tree for why our SaaS monthly revenue dropped last quarter"

**It needs from you:**
- SUBJECT (required): the 14% quarter-over-quarter drop in monthly recurring revenue for our B2B project-management SaaS
- PURPOSE (optional): diagnose the drop so we can assign each branch to an owner to investigate

**You get back:** a saved markdown logic tree (`mece-decomposition-mrr-drop-<date>.md`) with the chosen top-level cut and why it beat the alternatives, the full nested tree with each node's cutting logic, a per-level ME/CE check, and notes on how the branches map to workstreams.

**See a full worked example:** [mece-decomposition-mrr-drop-2026-06-19.md](mece-decomposition-mrr-drop-2026-06-19.md), the report this invocation produces.
