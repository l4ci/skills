# Example: vrio-analysis

**You invoke it with:**
> "Run a VRIO analysis on us to test whether our strengths are real, lasting advantages."

**It needs from you:**
- FIRM (required): Lumen Analytics, a B2B SaaS firm selling supply-chain forecasting software
- RESOURCES (optional): the proprietary demand-forecasting model, the ten-year shipment dataset, the data-science team's tacit know-how (if omitted, it surfaces candidates first)
- CONTEXT (optional): rivals are SAP and a well-funded startup; the question is which resources to invest behind for the next three years

**You get back:** a saved markdown report (`vrio-<firm-slug>-<date>.md`) with a summary table of each resource's V/R/I/O answers and competitive implication, per-resource detail with the inimitability mechanism, the strategic core, the fixable (valuable/rare/inimitable but unorganized) gaps, eroding edges, and strategic moves.
