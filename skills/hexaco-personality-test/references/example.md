# Example: hexaco-personality-test

**You invoke it with:**
> "Give me a HEXACO personality test, I want the Honesty-Humility one, not plain Big Five"

**It needs from you:**
- Nothing upfront, the skill interviews you / administers the questionnaire and you answer as you go.

**You get back:** a saved markdown report (`hexaco-personality-test-<date>.md`) with a summary table and text bars for all six factors, a per-factor deep dive with facet breakdowns, two to four notable profile interactions, reflection prompts, and caveats.

**See a full worked example:** [hexaco-personality-test-2026-06-19.md](hexaco-personality-test-2026-06-19.md), the report this invocation produces.
